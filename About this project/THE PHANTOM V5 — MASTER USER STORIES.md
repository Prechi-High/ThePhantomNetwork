THE PHANTOM V5 — MASTER USER STORIES

Status: Current Working User Stories (V5)
Scope: Includes only currently agreed functionality and corrections discussed throughout the project.

PLAYER / USER STORIES
Identity & Profile
US-001 — Automatic Account Creation

As a new user,

I want my account to be automatically created when I start the application,

So that I can immediately enter THE PHANTOM without registration friction.

US-002 — Automatic Telegram Integration

As a user,

I want Telegram onboarding to happen automatically,

So that I do not need to manually connect my Telegram account.

US-003 — Avatar Selection

As a user,

I want to select from predefined avatars,

So that I can personalize my identity.

US-004 — View Profile

As a user,

I want to view my profile,

So that I can see my progress and achievements.

US-005 — View Token History

As a user,

I want to view my token history,

So that I can track my gameplay performance.

US-006 — View Session History

As a user,

I want to view previous sessions,

So that I can review my performance.

US-007 — Earn Badges

As a user,

I want to unlock badges through progression,

So that I can display prestige.

CAMP STORIES
Camp Membership
US-008 — Automatic Camp Assignment

As a user,

I want to automatically join a camp during onboarding,

So that I immediately belong to a larger community.

US-009 — Referral Camp Assignment

As a referred user,

I want to automatically join the referring camp,

So that referrals strengthen existing communities.

US-010 — Default Camp Assignment

As a direct user,

I want to automatically join the default Phantom Camp,

So that I can participate immediately.

US-011 — Browse Camps

As a user,

I want to browse camps,

So that I can evaluate alternatives.

US-012 — Camp Switching

As a user,

I want to switch camps only after reaching an admin-defined level,

So that camp loyalty remains meaningful.

US-013 — View Camp Leaderboard

As a user,

I want to view camp rankings,

So that I can understand camp performance.

US-014 — View Camp Statistics

As a user,

I want to see camp performance data,

So that I understand my camp's influence.

US-017 — Leave Squad

US-018 — View Squad Information

As a user,

US-019 — View Squad Leaderboard

As a user,

So that I understand rankings.

US-020 — Earn Squad Tokens

As a squad member,

US-021 — Spend Squad Tokens

As a squad member,

So that squads can customize their identity.
