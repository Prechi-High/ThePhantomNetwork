Other systems subscribe.**\n\n### Before (Tight Coupling)\n```\nWheelComponent\n├─ Spin wheel\n├─ Play sound (embedded)\n├─ Animate tokens (embedded)\n└─ Update score (embedded)\n```\n\n### After (Clean Separation)\n```\nWheelComponent\n├─ Spin wheel\n└─ emit('spin.outcome.advance')\n    ├─ SoundSystem subscribes\n    ├─ AnimationSystem subscribes\n    ├─ HapticsSystem subscribes\n    └─ Analytics subscribes\n```\n\n**This changes everything.** Once the Event Bus exists:\n- Sound developer doesn't touch gameplay code\n- Animation developer doesn't touch gameplay code\n- Haptics developer doesn't touch gameplay code\n- Each system owns its own logic\n- Easy to test\n- Easy to extend\n\n---\n\n## What's Already Done\n\nYou inherit a solid foundation:\n\n✅ **Stores**\n-
